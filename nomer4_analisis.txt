Analisa program 
Pada program signup terdapat deklarasi sebuah variabel guna nantinya untuk
mengeksekusi sebuah program, pada prpgram sign up yang saya buat tidak ada sebuah class 
sigupnya dimana saya menggantinya dengan function validation yang berisikan username dan password, 
kemudan dibawahnya lagi terdapat sebuah percabangan if yang didalamnya terdapat ketentuan ketetuan
yang harus diinputkan dengan benar mengenai kriteria username, kondisi percabangan yang pertama jika 
email dan password kosong maka program tidak akan mengeksekusi dan memberitahukan bahwa username atau
passwordnya kosong, lalu program akan kembali untuk merintahkan mengisikan usernamenya. Dimulai dari
pengecekan input apakah email dan password sudah benar iinputkan atau belum, kemudian memfilter untuk email 
valid atau tidaknya, lalu kondisi percabangan if selanjutnya dimana mengkondisikan jika pasword yang 
diinputakan kurang dari 6 karakter maka sistem program secara otomatis salah, kemudian kondisi 
percabangan if berikutnya juga jika passowrd kedua pun tidak sesuai kriteria dimana harus mengandung 
satu huruf kapital dan juga angka jika tidak sesuai maka sistem akan membaca nya sebgai kesalahan dalam input, 
setelah itu masuk pada kondisi percabangan selanjutnya yaitu dimana masuk dalam penyimpanan nilai dari atribut 
email itu sendiri sebgai username yanga akan ditampilakn pada bagain Dashboard. Setelah fuction
validation php nya dibawah terdapat sebuah html dimana unutk mengatur jendela program nantinya 
supaya lebih tertata lagi. Kemudian pada bagain dashboard nya itu memanggil username yang telah 
diinputkan sebelumnya kemudian akan ditampilkan sebagai outputnya dan html akan mengeksekusi memnggil 
function username tersbut tadi.  Secara kesulurahan mungkin tidak jauh berbeda dengan program sigunup 
yang diberikan pada pertemuan 3 hanya tidak ada penggunaan sebuah class nya unutk eksekusi sistemnya dengan 
kriteria kriteria yang ditentukan pun sama
